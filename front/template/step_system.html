System

Use the following controls to change basic system settings regarding your DataTorrent installation. If there is an issue icon (!) next to a property name, it means that the property is preventing DataTorrent from functioning properly.

hadoop binary location This is the absolute path to the location of the hadoop executable on the node that the gateway is running on.
java version Version of the Java Runtime Environment that is available on the node that the gateway is running on.
gateway ip address The IP address of the gateway node that the gateway should listen on for requests.
gateway ip app address The IP address that other nodes in the cluster should use to reach the gateway.
! HDFS directory The location where all datatorrent stuff will go, including checkpointed data, recordings, etc
